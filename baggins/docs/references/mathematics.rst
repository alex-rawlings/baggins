mathematics package
===================

The mathematics package includes regularly-used mathematical operations, such as calculating the interquartile range, coordinate transforms, bootstrapping, or creating empirical cumulative distribution functions.

All methods and functions are called using::

   baggins.mathematics.XXX

where `XXX` is the method name.
